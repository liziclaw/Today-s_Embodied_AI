# 贡献指南

感谢对本项目的关注！

## 📖 阅读笔记格式

每篇论文阅读笔记应包含以下部分：

1. **论文信息** - 标题、作者、机构、发表、链接
2. **文章主要内容** - 背景、问题、方法、实验
3. **文章创新点** - 范式创新、技术创新、实验创新
4. **文章具体实验情况** - 设置、结果、消融实验
5. **与之前论文的关联** - 发展脉络、对比分析
6. **从中学到了什么** - 洞察、技术细节、实验启示

## 📝 文件命名

```
YYYY-MM-DD_论文简称_具身智能论文阅读.md
```

## 🔗 必须包含

- 论文 arXiv 链接
- PDF 直接链接
- 项目主页（如有）
- 代码仓库（如有）
